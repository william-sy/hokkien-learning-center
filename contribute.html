JSON Structure Guide

Dictionary entries are stored in data/dictionary.json as a simple array. Metadata (dialects, resources, tone charts) stays in data/content.json.

Field Reference

dialectId (required)

The dialect/variety this entry belongs to. Use one of these values:

  • "shared" - Shared / Cross-dialect
  • "quanzhou" - Quanzhou (泉州)
  • "zhangzhou" - Zhangzhou (漳州)
  • "xiamen" - Xiamen / Amoy (廈門)
  • "taiwanese" - Taiwanese
  • "sea_hokkien" - Southeast Asian Hokkien (General)
  • "malaysia_north" - Malaysian Hokkien (North)
  • "malaysia_central" - Malaysian Hokkien (Central)
  • "malaysia_south" - Malaysian Hokkien (South)
  • "singapore" - Singaporean Hokkien
  • "philippine" - Philippine Hokkien
  • "indonesian" - Indonesian Hokkien
  • "burmese" - Burmese Hokkien
  • "thai" - Thai Hokkien

english (required)

English translation or meaning. Keep it concise and clear.

Example: "Hello", "Thank you", "Where are you going?"

hanzi (optional)

Chinese characters (traditional). Use empty string "" if not applicable.

Example: "你好", "多謝", "好食"

poj (optional)

Pe̍h-ōe-jī romanization. Standard romanization system for Hokkien.

Example: "Lí-hó", "To-siā", "Hó-chia̍h"

tl (optional)

Tâi-lô (Taiwanese Romanization System). Official romanization in Taiwan.

Example: "Lí-hó", "To-siā", "Hó-tsia̍h"

tone (optional)

Tone numbers for each syllable, separated by hyphens.

Example: "2-2", "1-7", "2-4-3-2-7"

audioHint (optional)

Simplified pronunciation hint for learners without tone marks.

Example: "li-ho", "to-sia", "ho-tsiah"

audioUrl (optional)

URL to audio file (MP3, WAV, OGG). Leave as empty string "" if no audio available.

Example: "audio/hello.mp3", "https://example.com/audio/hello.mp3"

example (optional)

Example sentence showing the word/phrase in context.

Example: "Lí-hó! Góa sī sin-lâng. (Hello! I am new here.)"

tags (optional)

Array of tags for categorization and search. Use lowercase, relevant keywords.

Example: ["greeting", "beginner"], ["food", "restaurant"]

Suggested: greeting, politeness, food, question, daily, beginner, intermediate, advanced, family, travel, numbers, time, emotions

Complete entry example 
{
  "dialectId": "taiwanese",
  "english": "Thank you",
  "hanzi": "多謝",
  "poj": "To-siā",
  "tl": "To-siā",
  "tone": "1-7",
  "audioHint": "to-sia",
  "audioUrl": "",
  "example": "To-siā lí ê pang-chān.",
  "tags": ["politeness", "beginner"]
}

Audio: Audio buttons only appear when audioUrl has a value. Leave empty if unavailable.

Adding a resource entry 
{
  "title": "Hokkien Dictionary",
  "type": "Reference",

🐍 Python Contributor Tools

The scripts/ folder contains helper tools for power contributors. Run all commands from the repo root.

scripts/add_entry.py

Interactive CLI to add new words or phrases to data/dictionary.json with guided prompts.

python scripts/add_entry.py

scripts/ocr_dictionary.py

OCR pipeline for scanning physical Hokkien dictionaries via webcam or image file. Extracts entries automatically and flags uncertain results for review.

# Install dependencies first:
pip install opencv-python pytesseract Pillow
brew install tesseract   # macOS

# Run:
python scripts/ocr_dictionary.py

scripts/correct_poj.py

Fixes common Tesseract OCR misreads in POJ romanization. Run after OCR scanning, or use --dry-run to preview changes.

python scripts/correct_poj.py            # fix in-place
python scripts/correct_poj.py --dry-run  # preview only
python scripts/correct_poj.py --word sti # look up one word

scripts/test_parser.py

Quick sanity-check: runs the OCR parser against a saved ocr_output.txt file and prints the parsed entries.

python scripts/test_parser.py
